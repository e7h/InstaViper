# **InstaViper**
Simple Instagram password reset tool

## **About**
Tool to send password reset requests to Instagram accounts using email or username.

## **Features**
-  Reset via **Email**
-  Reset via **Username**
-  Colored interface
-  Fast & easy to use

## **Installation**

```bash
# Install requirements
pip install requests colorama

# Run the tool
python main.py
```

## **How to Use**

1. Run: `python main.py`
2. Choose option:
   - `1` - Email reset
   - `2` - Username reset  
   - `3` - Exit
3. Enter email/username

## **Requirements**
- Python 3.13+
- requests
- colorama
